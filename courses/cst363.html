This course provides balanced coverage of database use and design, focusing on relational databases. Students learn to design relational schemas, write SQL queries, access a database programmatically, and perform database administration. Topics also include algorithms and data structures for query evaluation, transaction processing, and newer technologies like XML, NoSQL, and Hadoop.

  • Apply newer database technologies like NoSQL, XML, and data warehousing.

üöÄ Final Project: Prescription Database Web Application

Our final project was a full-stack web application to manage prescriptions between doctors, patients, and pharmacies. It includes ER design, schema creation, SQL triggers, and user interface logic. The project emphasized real-world application of database design, SQL programming, and validation logic.
