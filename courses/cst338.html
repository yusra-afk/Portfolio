CST 338 - Software Design

📘 Course Overview

This is an intermediate-level programming course covering techniques for developing large-scale software systems using object-oriented programming. Coverage includes software development life cycle models, requirements analysis, and graphical user interface development.

🚀 Final Project: Ghetto Yelp

Project Description: Ghetto Yelp is a mobile application that allows users to view and create reviews for restaurants. Admins can manage the list of restaurants, while users can search, review, and browse entries. The project was built using Java with Room database integration, adhering to Android development best practices.

My Role: I developed the View All Restaurant and Add/Remove Restaurant activities (UI + back-end), created the Restaurant entity, managed GitHub issues and project deadlines, and presented features in the project video.
