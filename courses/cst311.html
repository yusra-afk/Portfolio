CST 311 - Intro to Computer Networks

CST 311 is an introduction to data communication technologies and computer networks. It covers concepts like Local and Wide Area Networks, TCP/IP protocols, routing, switching, socket programming, and modern networking challenges including security, peer-to-peer networks, and high-speed data transmission. Students use tools like Wireshark and develop Python socket applications.

üéØ Course Outcomes

  • Describe layered architectures and the TCP/IP model.
  • Use and explain the workings of common network applications and services.
  • Explain inter-layer relationships in TCP, UDP, IP, ARP, routing, and media access protocols.
  • Analyze and simulate network performance using tools and models.
  • Write basic client-server programs using Python sockets.
  • Understand emerging topics like wireless networking, multimedia transmission, and security.

üìå Notes

While this course did not include a final project submission, it involved intensive programming assignments, labs using Wireshark, and quizzes focused on real-world applications of networking principles.
