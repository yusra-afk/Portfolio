CST 383 — Introduction to Data Science

Overview

Course Overview

A hands-on introduction to data science using Python. Covered the full pipeline from data wrangling and exploratory analysis to visualization and introductory machine learning with scikit-learn.

Outcomes

Learning Outcomes

  • Use Python (pandas, NumPy) to clean and transform datasets.
  • Build exploratory visualizations with matplotlib and seaborn.
  • Apply regression, classification, and clustering techniques.
  • Communicate findings clearly through data storytelling.
